# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | Yes                |
| < 0.2.0 | No                 |

## Reporting a Vulnerability

If you discover a security vulnerability in kube-devops, please report it
responsibly using [GitHub private vulnerability reporting](https://github.com/stochasticquant/kubernetes-cluster-observer-and-controller-using-Rust/security/advisories/new).

**Please do not open a public issue for security vulnerabilities.**

When reporting, include:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to provide a fix or
mitigation within 7 days for confirmed issues.
